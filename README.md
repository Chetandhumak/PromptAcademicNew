| Feature | Description |
|---|---|
| 📚 **6 Learning Modules** | Foundations → Roles → CoT → Structure → API → Advanced |

## 🚀 Quick Start

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- [Git](https://git-scm.com/)

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/prompt-engineering-academy.git
cd prompt-engineering-academy
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### 4. Build for Production

```bash
npm run build
```

## 🤝 Contributing

Contributions are welcome! Here's how to get started:

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/add-new-module`
3. Make your changes and commit: `git commit -m 'feat: add module 7 - RAG prompting'`
4. Push to your branch: `git push origin feature/add-new-module`
5. Open a Pull Request

### Ideas for Contributions

- 📚 New learning modules (RAG, Agents, Tool Use, Vision)
- 🌍 Translations / internationalization
- 🎯 More quiz questions
- 🔌 Backend persistence (save progress to localStorage or a database)
